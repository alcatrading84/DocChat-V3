1. Introducción

DocChat es un asistente de IA que te permite chatear con tus documentos de forma natural. Sube un PDF, Word, Excel o cualquier documento, haz preguntas sobre su contenido y obtén respuestas en tiempo real.

🔒 100% Local
Tus datos nunca salen de tu PC
🔑 Sin API Keys
No necesitas OpenAI ni servicios externos
🎯 Sin LM Studio
Usa modelos GGUF locales
💰 Gratuito
Sin costos de uso ni suscripciones
📊 Multi-formato
20+ formatos de documentos
🚀 Streaming
Respuestas en tiempo real

🎯 ¿Para qué sirve DocChat?

  • 📄 Resumir documentos largos al instante
  • 🔍 Encontrar información específica en PDFs o Word
  • 📊 Extraer datos de tablas en Excel o CSV
  • 🌐 Traducir respuestas a inglés/español
  • 📚 Analizar múltiples documentos a la vez

2. Requisitos del Sistema

⚙️ Mínimos

  • Sistema operativo: Windows 10/11, Linux o macOS
  • RAM: 4 GB (8 GB recomendado)
  • Disco: 1 GB libre para el modelo local
  • Python: 3.10+ (solo si usas código fuente)

🧩 Opcionales

ComponentePara qué sirve
Tesseract OCRLeer PDFs escaneados (imágenes)
LM StudioUsar modelos más grandes vía API local
OpenAI API KeyModo Cloud con GPT-4o-mini

💡 Nota

DocChat funciona completamente sin internet. El modelo local se descarga automáticamente la primera vez que ejecutas el programa (~1 GB).

3. Instalación

📦 Opción A: Ejecutable (Recomendado)

  1. Descarga DocChat.exe desde GitHub Releases
  2. Haz doble clic para ejecutar
  3. En la primera ejecución, el modelo se descarga automáticamente (~1 GB)

✅ No necesitas instalar Python ni ninguna dependencia. El .exe incluye todo.

🐍 Opción B: Desde código fuente

# 1. Clonar el repositorio
git clone https://github.com/tuusuario/DocChat.git
cd DocChat

# 2. Instalar dependencias
pip install -r requirements.txt

# 3. Ejecutar
python run.py

4. Cómo Ejecutar

🖥️ Interfaz de Escritorio

5. Modos de Funcionamiento

DocChat tiene 3 modos seleccionables desde la interfaz, cada uno con sus ventajas:

🖥️ Modo LM Studio (por defecto)

Conecta con LM Studio corriendo en http://127.0.0.1:1234.

  • 💡 Ventajas: modelos más grandes, mayor calidad de respuesta
  • ⚠️ Requisito: tener LM Studio abierto con un modelo cargado
  • 🔧 Ideal para: respuestas detalladas y análisis profundos

☁️ Modo Cloud (OpenAI)

Usa la API de OpenAI (modelo gpt-4o-mini).

  • 💡 Ventajas: máxima calidad de respuestas
  • ⚠️ Requisito: API key válida de OpenAI
  • 🔧 Ideal para: tareas complejas que requieren precisión

🏠 Modo GGUF Local NUEVO

Usa llama-cpp-python para inferencia 100% local.

  • 💡 Ventajas: no requiere LM Studio ni internet
  • ⚠️ Requisito: el modelo se descarga automáticamente (~1 GB)
  • 🔧 Modelo por defecto: Qwen2.5-1.5B-Instruct Q4_K_M
  • 🔧 Alternativa ligera: Qwen2.5-0.5B (~350 MB)

6. Cómo Activar LM Studio

Sigue estos pasos para usar LM Studio con DocChat:

PasoAcción
1Descarga LM Studio desde lmstudio.ai e instálalo
2Abre LM Studio y busca un modelo (ej: Qwen2.5-7B-Instruct)
3Descarga el modelo desde la interfaz de LM Studio
4Haz clic en el botón "Start Server" (esquina inferior izquierda)
5Verifica que el servidor corre en http://127.0.0.1:1234
6En DocChat: selecciona "🖥️ LM" en el selector de modo superior

🔍 Verificar conexión

DocChat mostrará "🟢 Conectado" en la barra de estado cuando detecte LM Studio correctamente. Si ves "🔴 Desconectado", revisa que el servidor esté iniciado.

7. Guía de Uso

📎 Cargar Documentos

  1. Arrastra un PDF, Word o TXT al panel izquierdo (sidebar)
  2. O haz clic en la zona de carga para seleccionar archivos
  3. DocChat soporta 20+ formatos (PDF, DOCX, TXT, MD, HTML, CSV, XLSX, PPTX, código y más)

💬 Chatear con Documentos

  1. Carga uno o más documentos en la sidebar
  2. Escribe tu pregunta en el campo de texto inferior
  3. Presiona Enter o haz clic en el botón ▶
  4. Las respuestas aparecen en tiempo real (streaming)
  5. El modo RAG activado usa el contexto de los documentos

10. Solución de Problemas

🔴 "No se puede conectar con LM Studio"

  • Asegúrate de que LM Studio esté abierto
  • Verifica que el servidor esté iniciado (Start Server)
  • Confirma que el puerto sea 1234
  • Prueba cambiando al modo 🏠 GGUF que no requiere LM Studio

⏳ "El modelo no responde" o "tarda mucho"

  • La primera vez tarda ~1-2 min en descargar el modelo
  • Verifica que tengas suficiente RAM disponible (mín 4 GB)
  • Prueba con un modelo más pequeño (Qwen2.5-0.5B)
  • Cierra otros programas pesados

❌ "Error al cargar PDF"

  • Asegúrate que el PDF no esté protegido con contraseña
  • Para PDFs escaneados, instala Tesseract OCR
  • Prueba con un archivo más pequeño

💥 "El .exe no abre o se cierra"

  • Asegúrate de tener la última versión del .exe
  • Ejecuta desde una terminal para ver errores: DocChat.exe --debug
  • Desactiva el antivirus temporalmente (a veces bloquea .exe nuevos)
  • Verifica que no falten DLLs (VC++ Redistributable)
